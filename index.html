Vuforia Unity3D V4

Augmented reality project kit for Unity3D

Download this project as a .zip file Download this project as a tar.gz file

In order to use this project kit you need the following free software & services:

  • Unity3D
  • Java SE Development Kit - Needed to publish
  • Android SDK - Needed to publish to Android
  • Vuforia Account - Needed to run in Unity3D and publish to mobile

Watch here for information on self paced courses to learn how to make augmented reality applications.

Quickstart for testing in the Unity3D editor:

  1. Download & install Unity3D
  2. Register at Vuforia & generate a license key by clicking the "Develop" menu link
  3. Open the project kit in Unity3D and in the "Scenes" folder double click the "Main" scene and highlight in the "Hierarchy" tab the "ARCamera" then in the "Inspector" tab paste the license key into the "App License Key" field.
  4. Save the Unity3D project.
  5. Click the Unity3D "Play" icon and hold a dollar bill in front of your webcam to see the spider animate on it. The room should be decently lit.
